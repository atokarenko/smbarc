# AI Architect — платформа управления AI-трансформацией для SMB

## What This Is

B2B SaaS-платформа, которая помогает малому и среднему бизнесу системно управлять внедрением AI. Сканирует бизнес-процессы, строит карту AI-зрелости, определяет что автоматизировать, отслеживает эффект и управляет рисками. По сути — "AI-архитектор для предпринимателя": навигатор в мире AI-автоматизации.

## Core Value

Предприниматель получает ясную картину: где он сейчас с AI, что автоматизировать дальше, какой эффект это даст и какие риски нужно контролировать — вместо хаотичного внедрения инструментов.

### Out of Scope

- Реальные интеграции с CRM/HR/Finance системами — v2, для POC mock-данные
- Реальные AI-модули (HR automation, Finance assistant и т.д.) — v2+, для MVP каталог
- Мобильное приложение — web-first
- Billing/подписки — для POC не нужно
- OAuth/SSO — для POC простая авторизация или без неё

## Context

- Рынок: SMB внедряет AI хаотично, нет системного подхода к AI-стратегии
- Конкуренты: OpenAI делает модели, Microsoft — инструменты, никто не делает "AI-архитектора" для бизнеса
- Тренд: регуляция AI усиливается, скоро будут стандарты и аудит — платформа может стать инструментом compliance
- Монетизация: подписка $99-499/мес, 1000 клиентов = серьёзный оборот
- POC цель: рабочий локальный портал для демо ко-фаундерам, проверка идеи в действии

## Constraints

- **Timeline**: POC сегодня — минимально рабочий продукт для демо
- **Stack**: Full JS (Next.js / Node.js) — единый стек
- **AI**: Абстракция провайдера, локально через claude-max-proxy
- **Data**: Mock-данные для демо, без реальных интеграций
- **Deploy**: Локальный запуск
