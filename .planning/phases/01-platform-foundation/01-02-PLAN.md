       - On success: returns `{ status: 'connected', response: result.text, provider, model }`
       - On error: returns `{ status: 'error', message: String(error), provider, model }` with status 500
       - Wrap in try/catch for graceful failure when proxy is not running

         - ROI: moderate additional savings in compliance automation
         - Roadmap: expand fraud ML, add compliance automation, then advisory AI

       - **TechManufact Ltd.** (Manufacturing, advanced maturity ~78/100):

         - ROI: incremental optimization gains, quality control improvements
         - Roadmap: enhance predictive quality, add demand forecasting, autonomous logistics
