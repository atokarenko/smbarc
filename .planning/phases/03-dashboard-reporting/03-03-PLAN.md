2. Create `src/components/dashboard/pdf-export-button.tsx`:
   - "use client" component
   - Import dynamic from next/dynamic
   - Import Button from @/components/ui/button
   - Import Download icon from lucide-react
   - Import AssessmentResults type from @/lib/assessment/types
   - Props: `{ data: AssessmentResults; companyName: string }`
   - Dynamic import PDFDownloadLink with ssr: false:
     ```typescript
     const PDFDownloadLink = dynamic(
       () => import("@react-pdf/renderer").then((mod) => mod.PDFDownloadLink),
       { ssr: false, loading: () => <Button disabled><Download className="mr-2 h-4 w-4" />Loading...</Button> }
     );
     ```
   - Dynamic import AssessmentReport:
     ```typescript
     const AssessmentReport = dynamic(
       () => import("./pdf-report").then((mod) => mod.AssessmentReport),
       { ssr: false }
     );
     ```
   - IMPORTANT: Since PDFDownloadLink needs an actual React element as `document` prop, and dynamic components may not work directly as document prop, use a pattern that works:
     - Use a state-based approach: on mount, dynamically import both @react-pdf/renderer and the report component, then render
     - OR simpler: use `pdf()` function from @react-pdf/renderer to generate blob, then trigger download via URL.createObjectURL
     - The blob approach is more reliable with Next.js dynamic imports:
       ```typescript
       const handleExport = async () => {
         setLoading(true);
         const { pdf } = await import("@react-pdf/renderer");
         const { AssessmentReport } = await import("./pdf-report");
         const blob = await pdf(<AssessmentReport data={data} companyName={companyName} />).toBlob();
         const url = URL.createObjectURL(blob);
         const a = document.createElement("a");
         a.href = url;
         a.download = `${companyName}-health-report.pdf`;
         a.click();
         URL.revokeObjectURL(url);
         setLoading(false);
       };
       ```
   - Render a Button with Download icon, "Export PDF" text, onClick=handleExport
   - Show loading state while generating
