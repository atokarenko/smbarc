2. Create `src/lib/assessment/scoring.ts`:
   - `calculateDimensionScores(answers: Record<string, string>, sections: AssessmentSection[]): Record<string, number>` -- maps open-text answers to scores using heuristics (answer length, keyword presence for open-text; direct value for single-choice). Each dimension 0-100. Weights come from question.weight.
   - `calculateOverallScore(dimensionScores: Record<string, number>): number` -- weighted average of dimensions
   - `getMaturityLevel(score: number): { level: string, range: string }` -- maps score to CMMI level: Beginner (0-20), Developing (21-40), Intermediate (41-60), Advanced (61-80), Leader (81-100)

   - Scoring heuristic for open-text: Use answer length as a baseline signal (empty=0, short=25, medium=50, detailed=75, comprehensive=100), then adjust with keyword/phrase detection for domain relevance. This gives formula a reasonable baseline that AI will refine.
   - For single-choice questions: Map options to numeric scores directly.
